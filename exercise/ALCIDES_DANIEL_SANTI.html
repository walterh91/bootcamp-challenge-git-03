  1. Que es un sistema de control de versiones?

    Un sistema de control de versiones permite llevar un control de los cambios realizados en un repositorio de codigo, donde varias personas colaboran con modificaciones, arreglos o nuevas funcionalidades

  2. Cual es la diferencia entre git y github?

    Git es un sistema distribuido de control de versiones de codigo. Ademas "git" es el comando para manejar el versionado. Github, por su parte, es un servicio en la nube, que utiliza las potentes funcionalidades de git para hostear repositorios.
